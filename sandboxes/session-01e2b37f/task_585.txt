# Решение задачи по арифметической прогрессии

## Дано:
- Седьмой член прогрессии: \( a_7 = 8 \)
- Одиннадцатый член прогрессии: \( a_{11} = 12{,}8 \)

Требуется найти первый член \( a_1 \) и разность прогрессии \( d \).

## Решение:

### Шаг 1. Формула n-го члена арифметической прогрессии
Общий вид формулы для любого члена арифметической прогрессии:

\[
a_n = a_1 + d(n-1),
\]

где:
- \( a_n \) — n-й член прогрессии,
- \( a_1 \) — первый член прогрессии,
- \( d \) — разность прогрессии,
- \( n \) — номер члена.

### Шаг 2. Запишем уравнения для заданных членов

Для седьмого члена (\( n = 7 \)):

\[
a_7 = a_1 + d(7-1) = a_1 + 6d.
\]

По условию \( a_7 = 8 \), поэтому получаем первое уравнение:

\[
a_1 + 6d = 8. \quad (1)
\]

Для одиннадцатого члена (\( n = 11 \)):

\[
a_{11} = a_1 + d(11-1) = a_1 + 10d.
\]

По условию \( a_{11} = 12{,}8 \), поэтому получаем второе уравнение:

\[
a_1 + 10d = 12{,}8. \quad (2)
\]

### Шаг 3. Решаем систему уравнений

Имеем систему:

\[
\begin{cases}
a_1 + 6d = 8, \\
a_1 + 10d = 12{,}8.
\end{cases}
\]

Вычтем из уравнения (2) уравнение (1):

\[
(a_1 + 10d) - (a_1 + 6d) = 12{,}8 - 8.
\]

Упрощаем:

\[
a_1 + 10d - a_1 - 6d = 4{,}8,
\]
\[
4d = 4{,}8.
\]

Отсюда находим разность прогрессии \( d \):

\[
d = \frac{4{,}8}{4} = 1{,}2.
\]

### Шаг 4. Находим первый член \( a_1 \)

Подставим найденное значение \( d = 1{,}2 \) в уравнение (1):

\[
a_1 + 6 \cdot 1{,}2 = 8.
\]

Вычисляем:

\[
a_1 + 7{,}2 = 8.
\]

Переносим свободный член в правую часть:

\[
a_1 = 8 - 7{,}2 = 0{,}8.
\]

### Шаг 5. Проверка (для уверенности)

Проверим по формуле для одиннадцатого члена:

\[
a_{11} = a_1 + 10d = 0{,}8 + 10 \cdot 1{,}2 = 0{,}8 + 12 = 12{,}8.
\]

Результат совпадает с заданным значением \( a_{11} = 12{,}8 \), значит решение верное.

### Шаг 6. Ответ

\[
\boxed{a_1 = 0{,}8}
\]
\[
\boxed{d = 1{,}2}
\]

Первый член прогрессии равен \( 0{,}8 \), разность прогрессии равна \( 1{,}2 \).
