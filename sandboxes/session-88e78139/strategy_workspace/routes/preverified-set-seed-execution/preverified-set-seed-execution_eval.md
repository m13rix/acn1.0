# EVAL — Pre-verified Set Seed Execution

## Decision
**KEEP**

## Rank among siblings
**1 / 4**

## Question this depth is answering
What is the best broad family for a fast, stable, beginner-to-intermediate solo clear when the user is allowed to choose a seed?

## Evidence sources used
- Speedrun.com Minecraft: Java Edition guide index showing the category split between Random Seed and Set Seed.
- A current-version **Minecraft 1.21.11 set-seed glitchless run at 19:13**.
- TheGamer’s 1.21.11 seed article describing a very close village / portal / trial chamber style opener.
- Chunk Base Seed Map notes showing that pre-run structure scouting exists, but that it is an extra planning step, not a run-time advantage.

## Quantitative model
### Baseline
A representative current-version set-seed run exists at **19:13**. For a beginner-to-intermediate player, the realistic first-clear target is not world-record pace; it is a stable, repeatable finish.

### Expected time estimate
I model this family as:
- **setup time**: ~0 to 5 min if the seed is already chosen
- **run time**: ~20 to 35 min for a first stable personal clear
- **failure risk**: moderate but controllable, because the world opener is fixed

A rough first-clear expectation is therefore around **25 to 40 min total**, depending on skill and cleanup efficiency.

### Why this is strong
- Removing seed RNG eliminates the biggest early-time sink: restart fishing.
- A known opener lowers cognitive load and improves routing confidence.
- This family remains fast even before the user has speedrun-level mechanics.

## What-if checks
- **What if the user is weaker than expected?** This route still benefits from low variance and can be paired with a safer dragon finish.
- **What if the user is stronger than expected?** The route remains competitive because its route core is already close to optimal for stable clears.
- **What if a seed-map-selected world is even better?** That can improve the opener, but it rarely beats the combined simplicity of a pre-verified seed unless the improvement is large.

## Main unknowns
- The exact dragon-fight variant that best fits the user.
- Whether the user needs a few calibration drills before the first real attempt.

## Confidence
**High** that this is the best default root family.

## Why keep is justified
This is the cleanest match for the stated goal: fast, realistically executable, stable, and friendly to a beginner-to-intermediate solo player who is allowed to use a chosen seed. It has the best balance of speed and reliability among the depth-1 families.
