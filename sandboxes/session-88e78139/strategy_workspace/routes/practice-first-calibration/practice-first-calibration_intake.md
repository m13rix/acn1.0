# INTAKE — Practice-First Calibration

## Exact route question
Should the strategy spend a short amount of time measuring the user’s actual mechanics before choosing the final completion line?

## Why this route exists
- The user has not fully completed the game before.
- The user explicitly wants skill measured through tests instead of assumed.
- The best route depends on combat, inventory, movement, Nether navigation, and End-fight comfort.

## Assumptions
- Practice time does not count toward the run.
- Short drills are acceptable if they help choose the best route.
- The goal is still a fast clear, not a practice showcase.

## Dependencies
- Minimal but informative test battery.
- Honest interpretation of test results.
- Willingness to switch to a slightly safer route if a test shows weakness.

## What could break it
- The tests take too long.
- The tests are too generic and do not meaningfully predict actual route success.
- The user is already good enough that the practice time is just overhead.

## Information still needed
- Whether the user passes the opener / combat / End-fight tests cleanly.
- How much extra confidence the user needs before attempting the real run.
