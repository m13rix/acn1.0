# RESEARCH

## Key evidence gathered for depth-1 route comparison

- Speedrun.com shows Minecraft: Java Edition Any% Glitchless split by **Random Seed** and **Set Seed** seed types.
- A current-version set-seed run exists for **Minecraft 1.21.11** at **19:13**.
- A current-version random-seed run exists for **Minecraft 1.21.11** at about **51:52**.
- Chunk Base Seed Map supports Java 1.21 and can locate structures such as strongholds, bastions, villages, ruined portals, and more.
- Chunk Base warns that some structure positions are only approximate and some features can be inaccurate or missing depending on the target.
- Seed-map / structure-finder tools are good for pre-run scouting, but they add planning overhead and do not help execution skill directly.

## Practical interpretation

- For a beginner-to-intermediate solo player who is allowed to choose a seed, a **pre-verified set seed** is the cleanest low-variance route family.
- **Random-seed scout-heavy** is a real family, but it is clearly slower and more variable for this use case.
- **Practice-first calibration** matters because the user’s actual skill is unknown and can materially change route choice.
- **Seed-map scout-and-set** is useful if we need to tailor a world further, but it usually adds setup time and only wins if the tailored seed meaningfully improves the opener.
