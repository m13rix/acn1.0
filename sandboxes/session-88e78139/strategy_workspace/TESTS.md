# TESTS

## Minimum tests needed to resolve the skill-sensitive part of the strategy

1. **Opening execution test**
   - 5 minutes on a controlled or seeded world.
   - Goal: measure whether the user can loot a spawn area quickly, manage inventory, grab food, beds, and starter iron, and keep moving without losing track of items.
   - Pass signal: no major inventory mistakes, no panic deaths, and a clean first village/ruined-portal style opener.

2. **Basic combat / survival test**
   - 3 to 5 minutes in normal difficulty.
   - Goal: measure whether the user can fight off a small pack of common hostile mobs, keep health stable, and recover without dying.
   - Pass signal: safe shield usage or reliable kiting, no repeated damage spirals.

3. **End-fight handling test**
   - Short controlled drill.
   - Goal: measure whether the user can safely execute the intended dragon-kill method for the chosen route, especially if it uses beds.
   - Pass signal: the user can time placement / detonation and avoid self-kills.

4. **Bed timing drill**
   - 2 to 3 very short reps in a safe practice setup.
   - Goal: test whether the user can place a bed, create a blast shield if needed, and detonate without panic at a dragon perch analog.
   - Pass signal: at least one clean detonation and no repeated self-kills.

## How the tests are used

- If tests 1 and 2 pass cleanly, the strategy should bias toward the fastest stable set-seed route.
- If test 3 is shaky, prefer a safer dragon-fight variant over a faster but fragile one.
- If the user fails tests 1 or 2, elevate **Practice-First Calibration** and spend a little more time on drills before committing to the run.
- If the user fails the bed timing drill, kill the pure-bed finish and prefer the hybrid or bow-heavy fallback.
