## What-if checks

**What if Maxim rejects planners entirely?**
→ Score collapses. Mitigation: reframe as "strategy engine micro-session" rather than planner. Use intellectual framing: "You're optimizing your daily resource allocation."

**What if the planner becomes a compulsive system?**
→ This is the intellectualization loop. Mitigation: one action per day maximum. Never expand to a full time-blocking system.

**What if this works as a foundation for Journaling Protocol?**
→ Best case: the one-action planning feeds into the journaling entry ("what I did today, what I learned"). This is the optimal integration pattern — planner as input, journal as output.

## Main unknowns
1. Maxim's attitude toward daily planning.
2. Maxim's optimal productive window (morning/early afternoon assumed but not confirmed).
3. Whether the "ONE action" framing is sufficient.

## Confidence
**Moderate (65%)** — Addresses a real bottleneck (initiation) but has significant execution risk (planner resistance). The concept is sound but the psychological fit is less certain than Journaling Protocol.

## Why kill is justified
1. **Weaker metric alignment than Journaling Protocol:** Journaling directly targets a PRIMARY GOAL.md metric (>5 days/week). Visible-Action Planner only indirectly supports Knowledge Application Rate (secondary alignment).
2. **Higher resistance risk:** Maxim is a systems-builder who may view a daily planner as "beneath him" or "another system to maintain." Planner resistance is well-documented in executive function literature.
3. **Lower obsolescence clarity:** The transition from "planner-using" to "internally initiated" is harder to measure and enforce than journaling independence.
4. **Journaling Protocol absorbs core function:** The Journaling Protocol's "plan one small actionable step for tomorrow" prompt already includes action planning. A standalone planner is redundant.
5. **Higher maintenance burden:** Planning requires morning setup AND evening review. Journaling is one entry. Sustainability tie-breaker #4 favors Journaling.

## What would change the kill decision
- If Maxim expresses strong interest in daily planning.
- If Journaling Protocol fails (Maxim rejects journaling entirely).
- If we discover that Maxim's initiation failure is purely structural (time-of-day) rather than emotional — planner would be the precise tool.
