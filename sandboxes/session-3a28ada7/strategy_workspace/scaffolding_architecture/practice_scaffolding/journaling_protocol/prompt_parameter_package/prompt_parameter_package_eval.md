### Key metrics and targets
- **Prompt completion rate:** Baseline = 0%; Target = >70% within 2 weeks.
- **Prompt category preference:** Baseline = unknown; Target = identify 1-2 preferred categories by Week 3.
- **Emotional granularity score:** Baseline = broad labels ("bad", "stressed"); Target = >5 distinct emotion words used per entry by Week 4.

### Sensitivity analysis
- If Maxim rejects structured prompts entirely, score drops by ~1.5 (shift to open-ended "What's on your mind?" — still viable but loses targeted intervention power).
- If debugging framing feels like avoidance, score drops by ~0.8 (add more raw-emotion prompts).
- If prompt rotation is too fast, score drops by ~0.5 (slow rotation cadence).
- If Maxim already journals consistently, score increases by ~0.3 (shift from frequency-building to depth-building).

## What-if checks

**What if Maxim rejects all structured prompts?**
→ Fallback to single open-ended prompt: "What's on your mind?" Still maintains consistency metric. Structure can be reintroduced gradually once habit is stable.

**What if prompts trigger over-intellectualization?**
→ Add "body scan" or "physical sensation" prompts that force non-analytical responses. Include a "write the messiest possible version" constraint.

**What if prompt fatigue sets in?**
→ Accelerate rotation cadence OR introduce "Maxim's choice" day where he picks the prompt. Both increase autonomy.

**What if this works and prompts become unnecessary?**
→ Best case: Maxim generates his own prompts. The Prompt Parameter Package has explicit evolution toward self-prompting. This is the success path.

6. **Reversibility:** If a prompt category fails, it can be swapped instantly with zero cost.
