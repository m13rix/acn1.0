# INTAKE — Prompt Parameter Package

## Summary
The exact specification of journaling prompts for the Journaling Protocol: prompt categories, rotation schedule, prompt evolution over time, fallback prompts for low-energy/depleted days, and prompt count per session. Designed to target Maxim's perfectionism, all-or-nothing thinking, and emotional granularity deficits while leveraging his metacognitive strength.

## Question this route answers
"What are the exact journaling prompts — their categories, rotation, evolution schedule, and fallbacks — that will be delivered to Maxim during the Journaling Protocol?"

Position in family: Depth 5, parameter package 1 of 4 under Journaling Protocol. Siblings: Format & Delivery, Obsolescence & Transition, Tracking & Metrics.

## Assumptions
- Maxim has perfectionism in symbolic tasks (memory-confirmed) — prompts must explicitly normalize imperfection, skip days, and messy entries.
- Maxim's metacognition is strong — prompts can be more analytical than typical adolescent journaling prompts ("debugging" framing works).
- Maxim will respond better to prompts framed as "system diagnosis" (debugging consciousness) than traditional emotional reflection.
- Prompts should explicitly target the three pillars: Knowledge, Practice, Emotion.
- Prompt count per session: exactly 1 main prompt + 1 optional follow-up. More than that creates overload.
- Prompts rotate on a fixed schedule so Maxim does not become habituated.
- A "low-energy" fallback prompt set exists for days when Maxim is depleted (after 20:00, post-crash).

## Dependencies and permissions
- Permission to deliver 1 prompt per day via the delivery channel (Telegram, smartwatch, etc.).
- Permission to change prompts week-to-week based on what works and what does not.
- Permission to store prompt response data (content is private; metadata like completion rate is tracked).
- 3-5 minutes of Maxim's daily journaling session for prompt response.

## What this route enables
- **Perfectionism interruption:** Each prompt includes an "imperfection permission" element (e.g., "Write the messiest version possible").
- **Emotional granularity training:** Prompts move from broad ("How are you?") to precise ("Rate arousal 1-10 AND valence 1-10. Name the emotion with one exact word.").
- **All-or-nothing disruption:** Prompts target binary thinking with evidence-gathering questions ("What evidence disproves the idea that anything less than perfect is failure?").
- **Self-compassion growth:** Weekly self-compassion prompts adapted from the SCS-Youth framework (self-kindness, common humanity, mindfulness).
- **Practice-action bridge:** One prompt per week bridges journaling to concrete action ("What one small imperfect step can you take today?").
- **Prompt evolution over time:** Prompts become less structured, more open-ended as the protocol matures, supporting obsolescence.

## What could break it
- **Prompt fatigue:** Even with rotation, Maxim may become bored with structured prompts and start ignoring them.
- **Prompt perfectionism:** Maxim may try to write the "perfect" response to each prompt, defeating the purpose.
- **Too analytical, not emotional:** The debugging framing may over-intellectualize emotions — prompts must balance system-analysis with raw feeling.
- **Low-energy day resistance:** If the fallback prompt is still too demanding, Maxim may skip entirely rather than use the lighter option.
- **Crisis unsuitability:** On high-distress days, any prompt may be inappropriate. Protocol must allow an explicit "skip today" option.

## Main unknowns
1. Which prompt categories resonate most with Maxim (analytical vs. emotional vs. action-oriented)?
2. How quickly does Maxim habituate to prompt rotation? Optimal rotation cadence is unknown.
3. Whether Maxim prefers prompts that reference his own past entries (continuity) or fresh topics each time.
4. Whether the "debugging" framing actually works or feels like avoidance of real emotion.

## Evidence that matters most
- **Memory (perfectionism):** "Do it properly or not at all" pattern — prompts must actively resist this.
- **Memory (metacognition):** "Surprising amount of self-observation" — prompts can be more sophisticated than age-typical.
- **Web (perfectionism journaling):** Best practice 5 prompt types: identify all-or-nothing, challenge with evidence, celebrate wins, take imperfect action, self-compassion reframe.
- **Web (emotional granularity):** Moving from broad labels to precise emotion words improves regulation. Prompts should scaffold this.
- **Web (prompt fade):** Gradual reduction from structured to open-ended over weeks 1-8 supports independence.
- **Parent intake (obsolescence):** AI-prompted → self-prompted → independent. Prompt evolution schedule is key to this path.
