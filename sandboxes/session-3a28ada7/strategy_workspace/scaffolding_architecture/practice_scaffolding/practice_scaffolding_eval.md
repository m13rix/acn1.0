## What-if checks

**What if Maxim already has a stable practice routine?**
→ Memory suggests he does not: "every session requires fresh emotional activation." If he does, practice scaffolding shifts to optimization (earlier work window, better submission rhythm) rather than building from scratch.

**What if practice scaffolding becomes compulsive tracking?**
→ This is the intellectualization loop (GOAL.md failure state #2). Mitigation: minimum viable tracking. One checkbox per day, not a spreadsheet. The tracker itself must be designed to become obsolete.

**What if this works and completely reframes the problem?**
→ Best case: Maxim establishes self-sustaining rhythms that persist even after scaffolding is removed. He applies the same rhythm-building pattern to future challenges independently.

**What if one permission flips it?**
→ If Maxim rejects journaling, alternative practice channels: voice memos, smartwatch completion taps, or physical checklist. Still viable.

**What if stronger preparation makes it viable?**
→ Preparation: identify Maxim's optimal productive window, test 3 different journaling formats, establish a single visible-action protocol for school.

**What if this route is bad as a final route but excellent as an information-gathering opener?**
→ It is both. The information (when does Maxim actually work best? what triggers initiation failure?) is essential for all scaffolding layers. But it is also a viable final route because practice initiation is a genuine bottleneck.

## Main unknowns
1. Maxim's current journaling consistency (baseline).
2. Whether Maxim prefers digital or physical journaling.
3. Maxim's actual summer schedule — does he have commitments?
4. Whether Maxim has ever maintained a consistent routine for >1 week without external support.

## Confidence
**High (75%)** — Strong evidence that practice initiation is a real bottleneck. The risk is dependency on tracking tools, which is addressable through minimum-viable design.

## Why keep is justified (but as secondary priority)
1. **Real bottleneck:** Memory explicitly states the remaining bottleneck is "participation, oral answers, practice, emotional buy-in, and consistency." This is the primary problem after emotional regulation is addressed.
2. **Direct metric alignment:** Journaling consistency and knowledge-practice integration are primary GOAL.md metrics.
3. **Foundation for existential scaffolding:** Practice creates the action base on which meaning is built. Without practice, existential scaffolding is pure theory.
4. **Proof through consistency:** Completed action logs provide concrete evidence of change — feeds Maxim's proof-driven motivation.
5. **Early-day optimization:** Practice scaffolding directly addresses the evening depletion trap by shifting work to morning/early afternoon.

## Why this is rank 2, not rank 1
Emotional regulation must come first. Without emotional stability, practice scaffolding becomes a source of shame ("I can't even do the tracker"). The dependency chain is: emotional → practice → existential. Knowledge scaffolding is killed because it is not the bottleneck. Practice scaffolding is kept because it IS a bottleneck, but it is gated by emotional regulation.
