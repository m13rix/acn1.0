## What-if checks

**What if Maxim already has independent knowledge skills?**
→ Memory suggests he does — "extraordinary metacognitive abilities." If so, knowledge scaffolding is redundant. The real bottleneck is not knowledge generation but emotional activation to act on knowledge. This is the strongest argument for killing this route.

**What if emotional regulation scaffolding fails?**
→ Knowledge scaffolding becomes the intellectualization loop (GOAL.md failure state #2). Without emotional stability, Maxim will use strategy design as avoidance. This dependency makes knowledge scaffolding risky as a standalone layer.

**What if this works and completely reframes the problem?**
→ If Maxim internalizes problem decomposition, he may apply it to emotional regulation and practice — accelerating all layers. But this is contingent on emotional scaffolding succeeding first.

**What if one permission flips it?**
→ If Maxim rejects Socratic questioning, the entire knowledge scaffolding fails. There is no strong alternative pedagogy that fits his profile as well.

**What if stronger preparation makes it viable?**
→ Preparation: test whether Maxim already decomposes problems independently. If yes, this route is unnecessary. If no, it becomes viable.

**What if this route is bad as a final route but excellent as an information-gathering opener?**
→ It IS excellent for information gathering — testing whether Maxim's metacognition translates to independent strategy design. But this information can be gathered within the emotional regulation scaffolding layer (e.g., "design your own emotional regulation protocol" is a knowledge task embedded in an emotional context).

## Main unknowns
1. Maxim's current level of independent strategy ability (can he already decompose problems without AI?).
2. Whether Socratic questioning from an AI feels authentic or manipulative.
3. Whether knowledge gaps are actually the bottleneck (memory suggests they are not).

## Confidence
**Moderate (65%)** — The route is intellectually coherent and fits Maxim's cognitive style. But the evidence strongly suggests that knowledge is NOT the bottleneck — emotional regulation and practice initiation are. This makes knowledge scaffolding a secondary priority.

## Why kill is justified
1. **Not the bottleneck:** Memory explicitly states "knowledge gaps exist but are not the primary bottleneck; emotional resistance and meaning gaps dominate." Building scaffolding for a non-bottleneck is wasted effort.
2. **Risk of intellectualization loop:** Without emotional scaffolding first, knowledge scaffolding becomes another system to optimize instead of a tool for living (GOAL.md failure state #2).
3. **Redundancy with existing ability:** Maxim already has "extraordinary metacognitive abilities." He may not need scaffolding for what he already does well.
4. **Can be embedded in other layers:** Knowledge tasks (design your own protocol, decompose your own problem) can be embedded within emotional regulation and practice scaffolding. A standalone knowledge layer is unnecessary.
5. **Resource allocation:** 30-60 min/day budget is tight. Prioritizing emotional regulation and practice scaffolding yields higher returns.

## What we learn from killing this route
- Maxim's metacognitive ability may be sufficient without scaffolding.
- The real value of knowledge scaffolding is as an embedded feature within emotional and practice layers, not as a standalone layer.
- If emotional scaffolding succeeds, we can revisit whether a standalone knowledge layer is needed.
