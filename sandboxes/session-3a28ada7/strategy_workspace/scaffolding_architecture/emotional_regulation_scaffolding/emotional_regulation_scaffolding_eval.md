## What-if checks

**What if Maxim resists emotional regulation tools as "infantilizing"?**
→ Frame as "debugging protocol" not "therapy." Maxim already uses debugging metaphors for himself. "Your amygdala is flooding; the observer is the debugger." This intellectual framing may overcome ego resistance.

**What if emotional scaffolding works but Maxim never wants to remove it?**
→ This is the dependency trap (GOAL.md failure state #1). Mitigation: build graduation as proof. "You no longer need AI for this" is framed as a victory, not a loss. Use his proof-driven motivation.

**What if this works and completely reframes the problem?**
→ Best case: Maxim internalizes emotional regulation so well that the other scaffolding layers become easier to build. Emotional stability is the foundation for knowledge, practice, and existential work.

**What if one permission flips it?**
→ If smartwatch cannot deliver private notifications during school, delivery channel changes to phone or pre-scheduled music playlists. Still viable.

**What if stronger preparation makes it viable?**
→ Already viable. Preparation improvements: test smartwatch capabilities, pre-build 5 mood-mapped playlists, record 3 motivational speeches in Maxim's own voice.

**What if this route is bad as a final route but excellent as an information-gathering opener?**
→ It is both. The information gathered (how Maxim responds to obsolescence, what his baseline recovery time is) is essential for all other scaffolding layers.

5. **Window of opportunity:** Maxim is currently stable and positive. Building scaffolding during stability is easier than during crisis.
6. **Reversibility:** If dismantling is too fast, we can slow it. If too slow, we can accelerate. The schedule is adjustable.
