## Channel 2: WEB SEARCH (Professional Domain Knowledge)

### ISO Principle — Core Mechanism
- **FACT-W1:** The ISO principle (Iso = "same" in Greek) matches the client's current emotional/physiological state with music, then gradually shifts toward a target state through sequenced tracks.
- **FACT-W2:** Standard transition: 1-2 songs matching current mood → intermediate songs shifting by 5-10 BPM per song → 1-2 songs landing at target mood.
- **FACT-W3:** Total listening block: 20-30 minutes recommended for clinical settings; 15 minutes for daily self-guided use.
- **FACT-W4:** Stepwise progression avoids abrupt jumps — adjust tempo by ~5-10 BPM per song; maintain stylistic continuity.

### BPM-to-Mood Mapping (Evidence-Based Ranges)
| Mood/State | BPM Range | Description |
|------------|-----------|-------------|
| Anger / Agitation | 120-150 BPM | Fast, intense, aggressive rhythms |
| Anxiety / Stress | 90-120 BPM | Elevated, restless, tense |
| Sadness / Low-energy | 40-60 BPM | Slow, heavy, minor-key |
| Calm / Relaxation | 50-70 BPM | Slow, steady, peaceful |
| Happiness / Elation | 120-140 BPM | Upbeat, bright, major-key |
| Focus / Concentration | 80-100 BPM | Steady, rhythmic, minimal lyrics |

### Adolescent Music Intervention Protocols
- **FACT-W5:** Evidence-based adolescent protocols use 45-60 min therapist-led sessions 1-2x/week for 8-12 weeks; daily self-guided 10-20 min listening reinforces gains.
- **FACT-W6:** Tuned In program (Dingle et al., 2016): 98% attendance rate, significant emotion awareness improvements (p < 0.01) for at-risk adolescents.
- **FACT-W7:** Playlist structure: 5-8 tracks per session, each 2-4 minutes, organized in 3 sections (baseline → transition → target).
- **FACT-W8:** Music activates dopamine, modulates amygdala, regulates cortisol (Koelsch, 2009, 2010).
- **FACT-W9:** Empowering music significantly increases positive emotions post-stress (Fountain, 2023).
- **FACT-W10:** Music therapy for internalizing outcomes shows strongest evidence among adolescent interventions (Rodwin et al., 2022).

### Genre-Mood Effectiveness

### Smartwatch Music Control
- **FACT-W11:** Apple Watch can control Spotify playback via Now Playing app and MPRemoteCommandCenter (play/pause/skip). No native Spotify SDK for watchOS — deeper control requires iPhone routing.
- **FACT-W12:** Wear OS supports Spotify Android App Remote SDK — can start/pause/skip/queue tracks and select specific URIs. Requires OAuth scopes.
- **FACT-W13:** Spotify Web API Audio Features endpoint returns track tempo (BPM), key, mode, energy, valence — allows programmatic playlist building.
- **FACT-W14:** Automatic BPM-based selection is NOT built into Spotify — watch app must perform logic via Web API.
- **FACT-W15:** Must be within Bluetooth range; Spotify Premium required for full Connect control.

### Obsolescence / Fading Strategies
- **FACT-W16:** Effective fading: therapist-guided → adolescent selects matching music → adolescent times transitions → self-monitoring log replaces live guidance.
- **FACT-W17:** By month 3-4, adolescent expected to initiate full regulation cycle independently.
- **FACT-W18:** Self-generated cues (e.g., a specific "regulation song") replace external prompts.
- **FACT-W19:** Gradual withdrawal: weekly check-ins shift from modeling to brief reflection to autonomous use.
