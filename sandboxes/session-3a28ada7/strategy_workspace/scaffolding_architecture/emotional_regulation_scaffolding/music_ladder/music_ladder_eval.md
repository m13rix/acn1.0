## What-if checks

**What if Maxim resists mood playlists as "therapy for children"?**
→ Frame as "debugging your emotional state with audio tools" — intellectual framing. Or start with "what songs do you listen to when you feel X?" — organic, not clinical.

**What if music works but Maxim never wants to curate his own playlists?**
→ This is the dependency trap. Mitigation: build graduation as proof. "You no longer need AI for this" is framed as victory. Use his proof-driven motivation.

**What if this works and completely reframes the problem?**
→ Best case: Maxim internalizes somatic regulation so well that the other sub-components become easier. Emotional stability is the foundation.

**What if one permission flips it?**
→ If music platform access is denied, use local files or YouTube. Still viable.

**What if stronger preparation makes it viable?**
→ Already viable. Preparation improvements: map 5 existing playlists to mood states, test smartwatch audio, create 3 sample iso-principle playlists.

**What if this route is bad as a final route but excellent as an information-gathering opener?**
→ It is both. The information (music preferences, mood granularity, response to obsolescence) is essential for all other sub-components.

## Confidence
**High (85%)** — Strongest evidence base, best psychological fit, and clearest obsolescence path. Main risks are execution (music preference mapping) and ego resistance, both addressable.

## Why keep is justified
1. **Strongest evidence base:** Multiple RCTs, systematic reviews, and established clinical protocols (ISO principle, Tuned In program).
2. **Best psychological fit:** Sensual overexcitability + structured cognitive style + existing music engagement = near-perfect alignment.
3. **Lowest friction:** Music is already part of Maxim's daily life; this protocol adds structure to an existing behavior.
4. **Clearest obsolescence path:** AI-curated → Maxim-curated → independent use → retired. Each step is measurable.
5. **Fastest feedback loop:** Mood check before/after each listening block provides immediate data on effectiveness.
6. **Foundation for other sub-components:** Music calms the nervous system, making Maxim more receptive to speech and breathing protocols.
