# Notes Tool (Notion Integration)

Инструмент для работы с заметками в Notion через REST API.

## Возможности

- `notes.getAll(limit)` - получить список последних заметок (только названия)
- `notes.get(title)` - получить содержимое заметки по названию

## Настройка

### 1. Создать интеграцию в Notion

1. Перейдите на https://www.notion.so/my-integrations
2. Нажмите "+ New integration"
3. Дайте название (например, "ACN Agent")
4. Выберите workspace
5. Скопируйте "Internal Integration Token"

### 2. Подключить интеграцию к базе данных

1. Откройте нужную базу данных в Notion
2. Нажмите на "..." (три точки) в правом верхнем углу
3. Выберите "Connections" → "Connect to"
4. Найдите и выберите вашу интеграцию

### 3. Получить ID базы данных

URL базы данных выглядит так:
```
https://www.notion.so/{workspace}/{database_id}?v={view_id}
```

Скопируйте `database_id` (это строка между последним слэшем и знаком вопроса).

### 4. Настроить переменные окружения

Добавьте в `.env`:

```env
NOTION_API_KEY=secret_xxxxxxxxxxxxxxxxxxxxxxxxxx
NOTION_DATABASE_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

## Использование

### Получить список заметок

```javascript
// Получить 10 последних заметок (по умолчанию)
await notes.getAll();

// Получить 5 последних заметок
await notes.getAll(5);

// Получить 20 последних заметок
await notes.getAll(20);
```

### Получить содержимое заметки

```javascript
// По точному или частичному названию
await notes.get("Название заметки");

// Работает с частичным совпадением
await notes.get("Название");
```

## Требования к базе данных

База данных должна иметь свойство `Name` или `Title` (стандартное для Notion).

## Ограничения

- Максимум 100 заметок за один запрос (ограничение Notion API)
- Поиск по названию использует частичное совпадение (contains)
- При поиске возвращается первая найденная заметка

## Поддерживаемые типы блоков

Инструмент извлекает текст из следующих типов блоков Notion:
- Параграфы
- Заголовки (h1, h2, h3)
- Списки (маркированные, нумерованные)
- Цитаты
- Code blocks
- Callouts
- Toggle blocks
- И другие блоки с rich_text содержимым

Специальные блоки:
- Вложенные страницы: `[Вложенная страница: название]`
- Разделители: `---`
- Оглавление: `[Оглавление]`
