## Advanced PPTXML Usage

### Element Positioning Techniques
- Remember the slide dimensions: 10×5.6 units for 16:9 format
- Position is measured from the upper-left corner of the slide to the upper-left corner of elements
- When centering objects, calculate: x = (10-width)/2 and y = (5.6-height)/2
- Allow small margins (at least 0.5 units) from slide edges
- Ensure text boxes have adequate width and height to display content without crowding

### Z-Order Management
- Elements are layered in the order they appear in the code
- Place background elements (full slide backgrounds, decorative shapes) first
- Place content elements (text, images, charts) later so they appear on top
- Place highlights, callouts, or emphasis elements last

### Color Scheme Techniques
- Use 2-3 primary colors plus neutrals per presentation
- Corporate presentations: use conservative, professional colors
- Educational presentations: use engaging but not distracting colors
- Consider using theme colors with the scheme: prefix
- Create visual harmony through consistent color application

### Typography Best Practices
- Title slides: 40-48pt (size="40" to size="48")
- Slide headers: 32-36pt (size="32" to size="36")
- Body text: 18-24pt (size="18" to size="24")
- Caption text: 14-16pt (size="14" to size="16")
- Use sans-serif fonts for modern presentations (Arial, Calibri, Segoe UI)
- Use serif fonts for formal or traditional presentations (Georgia, Times New Roman)
- Maintain consistent font usage throughout the presentation

### Slide Design Patterns
1. **Title Slide Patterns:**
   - Central title with optional subtitle
   - Title with large background image
   - Title with decorative shape elements

2. **Content Slide Patterns:**
   - Left text, right image
   - Right text, left image
   - Top title, bottom split content
   - Full-width title with multi-column content

3. **Special Purpose Slides:**
   - Quote slides (large quote text, minimal other elements)
   - Comparison slides (side-by-side content)
   - Process slides (flowchart or sequential steps)
   - Data slides (charts with minimal supporting text)

### Design Enhancements
- Add subtle geometric shapes for visual interest
- Use semi-transparent shapes as text backgrounds for better readability
- Create consistent header/footer elements
- Use shape combinations to create custom visual elements
- Apply professional touch through precise spacing and alignment

# PPTXML: PowerPoint XML Markup Language

PPTXML is a concise markup language that allows you to create PowerPoint presentations using pptxgen.js through a simplified XML-like syntax.

## Basic Structure

```xml
<presentation 
  title="My Presentation" 
  author="Your Name" 
  company="Your Company" 
  subject="Presentation Subject" 
  revision="1" 
  layout="16x9">
  
  <!-- Slides go here -->
  <slide bg="#FFFFFF" color="#000000">
    <!-- Slide elements go here -->
  </slide>
  
</presentation>
```

## Presentation Properties

```xml
<presentation 
  title="Title" 
  author="Author Name" 
  company="Company Name" 
  subject="Subject" 
  revision="1" 
  layout="16x9|4x3|CUSTOM|etc" 
  rtl="true|false"
  theme-headFont="Arial"
  theme-bodyFont="Calibri">
  <!-- When using custom layout -->
  <layout name="myLayout" width="10" height="7.5" />
  
  <!-- Slides go here -->
</presentation>
```

## Slide Properties

```xml
<slide 
  bg="#FFFFFF|path/to/image.jpg|data:image/png;base64,..." 
  bg-opacity="0.5" 
  color="#000000|scheme:accent1" 
  hidden="true|false"
  master="MASTER_NAME">
  
  <slideNumber x="0.5" y="7" font="Arial" size="12" color="#333333" />
  
  <!-- Slide elements go here -->
</slide>
```

## Text Elements

```xml
<text 
  value="Single text value" 
  x="1" y="1" w="8" h="1.5" 
  align="left|center|right" 
  valign="top|middle|bottom"
  font="Arial" size="24" color="#333333"
  bold="true|false" italic="true|false" underline="true|false"
  bullet="true|false" indent="0" 
  hyperlink="https://example.com"
  glow="true" shadow="true"
  rotate="45">
</text>

<!-- Multi-formatted text -->
<text x="1" y="1" w="8" h="2">
  <span value="Regular text" />
  <span value="Bold text" bold="true" />
  <span value="Colored text" color="#FF0000" />
</text>

<!-- Text with bullets -->
<text x="1" y="1" w="8" h="3" bullet="true">
  <span value="First bullet point" />
  <span value="Second bullet point" />
  <span value="Third bullet point" />
</text>

<!-- Advanced bullets -->
<text x="1" y="1" w="8" h="3">
  <span value="Level 1 bullet" bullet="true" indent="0" />
  <span value="Level 2 bullet" bullet="true" indent="1" />
  <span value="Level 3 bullet" bullet="true" indent="2" bullet-type="number" />
</text>
```

## Images

```xml
<image 
  path="path/to/image.jpg|https://example.com/image.jpg" 
  data="base64_encoded_data" 
  x="1" y="1" w="4" h="3"
  alt="Alternative text"
  flip-h="true|false" flip-v="true|false"
  rotate="45"
  rounding="true|0.2"
  transparency="0.5">
  
  <!-- Optional sizing element -->
  <sizing type="crop|contain|cover" w="4" h="3" x="0" y="0" />
</image>
```

## Shapes

```xml
<shape 
  type="RECTANGLE|OVAL|TRIANGLE|LINE|etc" 
  x="1" y="1" w="4" h="3"
  fill="#FF0000|scheme:accent1" 
  fill-opacity="0.5"
  line="#000000" line-size="1" line-dash="solid|dash"
  rotate="45"
  flip-h="true|false" flip-v="true|false"
  hyperlink="https://example.com"
  text="Text inside shape"
  text-align="center"
  text-valign="middle"
  text-color="#FFFFFF"
  text-font="Arial"
  text-size="14"
  shadow="true">
</shape>
```

## Tables

```xml
<table x="1" y="1" w="8" h="4" 
       font="Arial" size="12" color="#333333"
       fill="#F7F7F7"
       border="1px solid #000000"
       align="center" valign="middle"
       autoPage="true" autoPageRepeatHeader="true" autoPageHeaderRows="1">
       
  <!-- Define column widths (optional) -->
  <colW>2,3,3</colW>
  
  <!-- Define row heights (optional) -->
  <rowH>0.5,0.75,0.75,0.75</rowH>
  
  <!-- Header row -->
  <row fill="#DDDDDD" bold="true">
    <cell value="Header 1" />
    <cell value="Header 2" />
    <cell value="Header 3" />
  </row>
  
  <!-- Data rows -->
  <row>
    <cell value="Cell 1,1" fill="#EEEEEE" />
    <cell value="Cell 1,2" />
    <cell value="Cell 1,3" />
  </row>
  
  <row>
    <cell value="Cell 2,1" />
    <cell value="Cell 2,2 with formatting">
      <span value="Bold text" bold="true" />
      <span value="Normal text" />
    </cell>
    <cell value="Cell 2,3" border="1px solid #FF0000" />
  </row>
</table>
```

## Charts

```xml
<chart type="bar|line|pie|scatter|area|radar|etc" 
       x="1" y="1" w="8" h="5"
       title="Chart Title" 
       title-color="#333333" title-size="16"
       show-legend="true" legend-pos="right|bottom|left|top" 
       legend-color="#333333" legend-size="10"
       show-data-table="true" show-labels="true">
       
  <!-- Chart data series -->
  <series name="Series 1" color="#4472C4">
    <point label="Jan" value="8.2" />
    <point label="Feb" value="3.2" />
    <point label="Mar" value="1.4" />
    <point label="Apr" value="6.2" />
  </series>
  
  <series name="Series 2" color="#ED7D31">
    <point label="Jan" value="2.5" />
    <point label="Feb" value="5.3" />
    <point label="Mar" value="5.7" />
    <point label="Apr" value="3.9" />
  </series>
  
  <!-- Optional axis configuration -->
  <catAxis title="Months" show-grid-lines="true" />
  <valAxis title="Values" min="0" max="10" major-unit="2" />
</chart>
```

## Media Elements

```xml
<media type="audio|video|youtube" 
       path="path/to/file.mp4"
       data="base64_encoded_data"
       link="https://www.youtube.com/watch?v=VIDEO_ID"
       x="1" y="1" w="8" h="5"
       cover="path/to/cover.jpg">
</media>
```

## Speaker Notes

```xml
<notes>These are speaker notes for the current slide.</notes>
```

## Color Schemes

The `scheme:` prefix can be used in any color attribute to reference PowerPoint theme colors:
- `scheme:text1`
- `scheme:text2`
- `scheme:background1`
- `scheme:background2`
- `scheme:accent1` through `scheme:accent6`

Example:
```xml
<text value="Accent colored text" color="scheme:accent1" />
```
