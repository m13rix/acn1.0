# Search Tool

Инструмент для поиска в интернете с двумя режимами работы.

## Возможности

- 🔍 **getAnswer** - Получение прямых ответов через Exa Answer API
- 🌐 **webSearch** - Веб-поиск через Gemini с инструментом Google Search

## Режимы работы

### 1. Exa Answer (getAnswer)

Использует Exa Answer API для получения прямых, хорошо проработанных ответов на вопросы.

**Преимущества:**
- Качественные, подробные ответы
- Специализирован на ответах на вопросы
- Быстрая обработка

**Когда использовать:**
- Нужен прямой ответ на конкретный вопрос
- Требуется информация по определенной теме
- Поиск фактов и определений

### 2. Gemini Web Search (webSearch)

Использует Gemini Flash Lite Latest с встроенным инструментом Google Search для поиска и анализа информации.

**Преимущества:**
- Полноценный Google Search
- Анализ и синтез результатов
- Быстрая обработка и высокое качество
- Интеграция с Google поиском

**Когда использовать:**
- Нужен широкий поиск по интернету
- Требуется анализ нескольких источников
- Поиск актуальной информации
- Поиск свежих данных и новостей

## Настройка

### 1. Получение API ключей

#### Exa API Key

1. Перейдите на [exa.ai](https://exa.ai/)
2. Зарегистрируйтесь или войдите в аккаунт
3. Получите API ключ в настройках

#### Gemini API Key

1. Перейдите на [Google AI Studio](https://aistudio.google.com/apikey)
2. Войдите с помощью Google аккаунта
3. Создайте новый API ключ

### 2. Установка переменных окружения

Создайте или обновите файл `.env` в корне проекта:

```env
# Exa API для getAnswer
EXA_API_KEY=your_exa_api_key_here

# Gemini API для webSearch
GEMINI_API_KEY=your_gemini_api_key_here
```

**⚠️ Важно:** Не публикуйте файл `.env` в публичные репозитории!

### 3. Установка зависимостей

```bash
npm install
```

## Использование

### Получение ответа (Exa)

```javascript
// Прямой ответ на вопрос
const answer = await search.getAnswer('Что такое квантовая физика?');

// Поиск информации
const info = await search.getAnswer('Последние новости о ИИ');

// Определения
const definition = await search.getAnswer('Определение термина машинное обучение');
```

### Веб-поиск (Gemini)

```javascript
// Широкий поиск по теме
const results = await search.webSearch('лучшие практики разработки на JavaScript');

// Поиск актуальной информации
const news = await search.webSearch('новости технологий 2025');

// Исследование темы
const research = await search.webSearch('как работают нейронные сети');
```

## Примеры запросов

### getAnswer (Exa)

```javascript
// Исторический вопрос
await search.getAnswer('Кто изобрел электричество?')

// Научный вопрос
await search.getAnswer('Как работает фотосинтез?')

// Техническая информация
await search.getAnswer('Что такое REST API?')
```

### webSearch (Gemini)

```javascript
// Поиск туториалов
await search.webSearch('tutorial react hooks 2025')

// Сравнение технологий
await search.webSearch('Next.js vs Remix сравнение')

// Актуальные данные
await search.webSearch('погода в Москве сейчас')
```

## Особенности

### Потоковый вывод

Оба режима поддерживают потоковый вывод результатов:
- Результаты выводятся в консоль по мере получения
- Серый текст показывает промежуточные результаты
- Финальный результат сохраняется и возвращается

### Обработка ошибок

Инструмент автоматически обрабатывает ошибки:
- Отсутствие API ключей
- Ошибки сети
- Ошибки API

Все ошибки логируются и возвращаются в понятном формате.

## Ограничения

### Exa Answer API

- Требуется валидный API ключ
- Возможны лимиты на количество запросов (зависит от тарифного плана)
- Работает только на английском и некоторых других языках

### Gemini API

- Требуется валидный API ключ
- Бесплатный tier имеет ограничения по количеству запросов
- Использует модель `gemini-flash-lite-latest` с Google Search

## Сравнение режимов

| Характеристика | getAnswer (Exa) | webSearch (Gemini) |
|----------------|-----------------|---------------------|
| **Скорость** | Быстрая | Очень быстрая |
| **Качество ответов** | Очень высокое | Высокое |
| **Широта поиска** | Сфокусирован | Широкий (Google Search) |
| **Анализ источников** | Автоматический | С синтезом |
| **Актуальность** | Высокая | Очень высокая |
| **Лучше для** | Прямые ответы | Исследование тем |

## Устранение неполадок

### Ошибка "EXA_API_KEY не установлен"

Убедитесь, что переменная окружения `EXA_API_KEY` установлена в файле `.env`

### Ошибка "GEMINI_API_KEY не установлен"

Убедитесь, что переменная окружения `GEMINI_API_KEY` установлена в файле `.env`

### Ошибка авторизации API

1. Проверьте правильность API ключей
2. Убедитесь, что ключи активны и не истекли
3. Проверьте квоты и лимиты вашего аккаунта

### Медленная работа

- Exa обычно отвечает быстро
- Gemini с Google Search работает очень быстро, но может занять время для сложных запросов
- Проверьте качество интернет-соединения
