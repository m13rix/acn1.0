# Exa Tool

Инструмент для поиска в интернете с использованием Exa Answer API и Exa Search API.

## Возможности

- 🔍 **getAnswer** - Получение прямых ответов через Exa Answer API
- 🌐 **webSearch** - Веб-поиск через Exa Search API с возвратом форматированных результатов

## Режимы работы

### 1. Exa Answer (getAnswer)

Использует Exa Answer API для получения прямых, хорошо проработанных ответов на вопросы.

**Преимущества:**
- Качественные, подробные ответы
- Специализирован на ответах на вопросы
- Быстрая обработка

**Когда использовать:**
- Нужен прямой ответ на конкретный вопрос
- Требуется информация по определенной теме
- Поиск фактов и определений

### 2. Exa Search (webSearch)

Использует Exa Search API для поиска в интернете и возвращает форматированную строку результатов, готовую для использования в LLM моделях.

**Преимущества:**
- Прямой доступ к результатам поиска
- Форматированный вывод результатов
- Глубокий поиск с контекстом и текстом страниц
- Результаты готовы для передачи в LLM модели

**Когда использовать:**
- Нужен доступ к сырым результатам поиска
- Требуется форматированная строка результатов для дальнейшей обработки
- Поиск с глубоким анализом страниц

## Настройка

### 1. Получение API ключа

#### Exa API Key

1. Перейдите на [exa.ai](https://exa.ai/)
2. Зарегистрируйтесь или войдите в аккаунт
3. Получите API ключ в настройках

### 2. Установка переменных окружения

Создайте или обновите файл `.env` в корне проекта:

```env
# Exa API для getAnswer и webSearch
EXA_API_KEY=your_exa_api_key_here
```

**⚠️ Важно:** Не публикуйте файл `.env` в публичные репозитории!

### 3. Установка зависимостей

```bash
npm install
```

## Использование

### Получение ответа (Exa Answer)

```javascript
// Прямой ответ на вопрос
const answer = await exa.getAnswer('Что такое квантовая физика?');

// Поиск информации
const info = await exa.getAnswer('Последние новости о ИИ');

// Определения
const definition = await exa.getAnswer('Определение термина машинное обучение');
```

### Веб-поиск (Exa Search)

```javascript
// Поиск с возвратом форматированных результатов
const results = await exa.webSearch('лучшие практики разработки на JavaScript');
// Результаты возвращаются в виде форматированной строки:
// Search results for: "лучшие практики разработки на JavaScript"
//
// [1] Название статьи
// URL: https://example.com/article
// Content: Текст статьи...
// Context: Контекст...
//
// [2] Следующий результат...

// Поиск актуальной информации
const news = await exa.webSearch('новости технологий 2025');

// Исследование темы
const research = await exa.webSearch('как работают нейронные сети');
```

## Формат результатов webSearch

Метод `webSearch` возвращает форматированную строку в следующем формате:

```
Search results for: "ваш запрос"

[1] Название страницы
URL: https://example.com/page
Content: Текст страницы (первые 500 символов)...
Context: Контекстная информация...

[2] Следующая страница
URL: https://example.com/page2
Content: Текст страницы...
...
```

Этот формат готов для передачи в LLM модели для дальнейшей обработки.


## Особенности

### Потоковый вывод

Метод `getAnswer` поддерживает потоковый вывод результатов:
- Результаты выводятся в консоль по мере получения
- Серый текст показывает промежуточные результаты
- Финальный результат сохраняется и возвращается

### Форматирование результатов

Метод `webSearch` автоматически форматирует результаты поиска:
- Каждый результат содержит заголовок, URL, контент и контекст
- Результаты пронумерованы для удобства
- Формат готов для передачи в LLM модели

### Обработка ошибок

Инструмент автоматически обрабатывает ошибки:
- Отсутствие API ключей
- Ошибки сети
- Ошибки API

Все ошибки логируются и возвращаются в понятном формате.

## Ограничения

### Exa API

- Требуется валидный API ключ
- Возможны лимиты на количество запросов (зависит от тарифного плана)
- Работает только на английском и некоторых других языках

## Сравнение режимов

| Характеристика | getAnswer (Exa Answer) | webSearch (Exa Search) |
|----------------|------------------------|------------------------|
| **Скорость** | Быстрая | Средняя |
| **Качество ответов** | Очень высокое | Зависит от обработки |
| **Формат вывода** | Готовый ответ | Форматированная строка результатов |
| **Глубина поиска** | Сфокусирован | Глубокий (deep search) |
| **Контекст** | Автоматический | Включен в результаты |
| **Лучше для** | Прямые ответы | Получение сырых результатов |

## Устранение неполадок

### Ошибка "EXA_API_KEY не установлен"

Убедитесь, что переменная окружения `EXA_API_KEY` установлена в файле `.env`

### Ошибка авторизации API

1. Проверьте правильность API ключа
2. Убедитесь, что ключ активен и не истек
3. Проверьте квоты и лимиты вашего аккаунта

### Медленная работа

- Exa Answer обычно отвечает быстро
- Exa Search может занять время для глубокого поиска
- Проверьте качество интернет-соединения
