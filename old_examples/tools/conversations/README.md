# Conversations Tool

Инструмент для поиска транскрипций разговоров за последние 2 дня.

## Функции

### `conversations.search(query)`

Поиск во всей базе данных транскрипций разговоров с использованием семантического поиска.

**Параметры:**
- `query` (string, обязательный) - описание ситуации для поиска
  - Пример: "13 говорит своё мнение по поводу демо нового робота"
  - Пример: "обсуждение проекта с клиентом"
  - Пример: "вопросы по техническим деталям"

**Возвращает:**
- Строку с результатами поиска, включающую:
  - Количество найденных транскрипций
  - До 3 наиболее релевантных результатов
  - Временные метки
  - Процент схожести
  - Текст транскрипции
  - Количество файлов (если доступно)

**Особенности:**
- Использует API эндпоинт: `https://telos-database-6tvke.sevalla.app/v1/audio/transcripts/search`
- Автоматически ограничивает результаты до 3 самых релевантных
- Транскрипции могут содержать неточности, особенно в именах
- Рекомендуется проверять информацию по смыслу

**Пример использования:**
```javascript
// Поиск обсуждения конкретной темы
const results = await conversations.search("13 говорит своё мнение по поводу демо нового робота");

// Поиск технических обсуждений
const techResults = await conversations.search("обсуждение архитектуры системы");

// Поиск по участникам
const participantResults = await conversations.search("Алексей задает вопросы по проекту");
```

## Технические детали

- Использует семантический поиск с эмбеддингами
- Минимальная схожесть: 0.0 (настраивается на сервере)
- Максимальное количество результатов: 3
- Поддерживает поиск по контексту и смыслу, а не только по точным совпадениям
