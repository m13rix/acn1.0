# Telos Spark - Development Guide

## Hot Reload (Редактирование в реальном времени)

Для разработки интерфейса с автоматическим обновлением страницы:

### Шаг 1: Запустите Backend сервер (в одном терминале)
```bash
npm start
```
Это запустит сервер на `http://localhost:1314` с API и WebSocket.

### Шаг 2: Запустите Vite Dev Server (в другом терминале)
```bash
cd client
npm run dev
```
Это запустит фронтенд на `http://localhost:5173` с hot reload.

### Шаг 3: Откройте браузер
Откройте `http://localhost:5173` (НЕ 1314!)

Теперь любые изменения в файлах в папке `client/src/` будут автоматически применяться в браузере без перезагрузки страницы.

---

## Production Build

Для финальной сборки (когда всё готово):
```bash
cd client
npm run build
```
Затем запустите `npm start` - сервер будет отдавать собранные файлы из `client/dist/`.
